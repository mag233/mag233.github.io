# mag233.github.io

This is a simple GitHub Pages blog using Jekyll and the minima theme.

## Structure
- `_config.yml`: Site configuration
- `index.md`: Home page
- `_posts/`: Blog posts (Markdown files)

## Usage
Push this repository to your `mag233.github.io` GitHub repository. GitHub Pages will automatically build and publish your site.

For more info, see [GitHub Pages documentation](https://pages.github.com/).
