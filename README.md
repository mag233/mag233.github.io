# mag233.github.io

This is a simple GitHub Pages blog using Jekyll and the minima theme.

## Structure
- `_config.yml`: Site configuration
- `index.md`: Home page
- `_posts/`: Blog posts (Markdown files)

## Usage
Push this repository to your `mag233.github.io` GitHub repository. GitHub Pages will automatically build and publish your site.

## Writing workflow options
- Browser helper (no install): visit `/write/` to generate a ready-to-commit post file (enable nav link by setting `show_write: true` in `_config.yml`).
- Local helper (PowerShell): run `powershell -ExecutionPolicy Bypass -File scripts/new-post.ps1` to create a new post with current time and template.

For more info, see [GitHub Pages documentation](https://pages.github.com/).
